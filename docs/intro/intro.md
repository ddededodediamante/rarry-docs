# Introduction

Rarry is a web-based platform that allows to create games or projects using visual block-coding, inspired by [Scratch](https://scratch.mit.edu/).

## Features

- **Visual Block Editor:** Build logic and projects without typing code.
- **Fast & Quick:** Converts blocks into code that runs blazingly fast.
- **Learning Friendly:** Learn programming concepts like loops, events, and conditionals.
- **Custom Extensions:** Expand with your own custom blocks using JavaScript.

## Get Started

1. Go to the [Rarry](https://scratch.mit.edu/) website.
2. Click the "Launch the Editor" button.
3. Start dragging blocks and bring your ideas to life!
